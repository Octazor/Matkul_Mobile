Deskripsi
Aplikasi sederhana untuk menyapa pengguna berdasarkan input nama, dilengkapi dengan fitur toggle untuk mengubah tema (Dark/Light Mode).

Cara Menjalankan
1. Clone repository ini atau download ZIP.
2. Buka Android Studio.
3. Pilih File > Open dan cari folder project ini.
4. Tunggu indexing selesai.
5. Klik tombol Run (Play hijau) di toolbar atas.
